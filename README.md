## 🚀 Getting Started

1. Install PostgreSQL on your system
2. Create a new database: `CREATE DATABASE learn_sql;`
3. Run the setup scripts in order
4. Follow through each tutorial section
5. Practice with the provided exercises

## 📝 Prerequisites

- PostgreSQL 12+ installed
- Basic understanding of databases
- A SQL client (pgAdmin, DBeaver, or psql command line)

## 💡 Learning Path

1. Start with BASIC section to understand fundamentals
2. Move to INTERMEDIATE to master queries and joins
3. Advance to ADVANCED for professional-level skills
4. Run scripts and practice exercises regularly
